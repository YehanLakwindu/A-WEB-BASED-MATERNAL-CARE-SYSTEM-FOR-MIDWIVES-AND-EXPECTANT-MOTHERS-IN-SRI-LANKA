# Selenium Automation Exercise Project

## Overview

This project demonstrates the use of Selenium WebDriver to automate browsing tasks on the "Automation Exercise" website. The tasks include opening the website, registering a user, logging in, and performing a product search. The project is built using Java and Selenium WebDriver.

## Prerequisites

Before running this project, ensure the following:

- **Java** is installed on your system.
- **Selenium WebDriver** dependency is added to your project. You can use Maven or download the JAR files directly from the [Selenium website](https://www.selenium.dev/downloads/).
- **ChromeDriver** is installed on your machine and its path is specified in the code.
- **IDE** (e.g., IntelliJ IDEA, Eclipse) for running Java code.

## Files and Functionality

1. **OpenWebsite.java**:

   - Opens the "Automation Exercise" website.
   - Prints a success message when the website is opened.
   - Closes the browser after the operation is completed.

2. **RegisterUser.java**:

   - Navigates to the "Signup / Login" page.
   - Fills out the registration form with user details such as name, email, and password.
   - Submits the registration form.
   - Prints a success message upon successful registration.
   - Closes the browser after completion.

3. **UserLogin.java**:

   - Navigates to the "Signup / Login" page.
   - Logs in with predefined user credentials.
   - Prints a success message upon successful login.
   - Closes the browser after completion.

4. **SearchWinterTop.java**:
   - Navigates to the "Products" page.
   - Performs a search for the product "Winter Top."
   - Prints a success message when the search results are displayed.
   - Closes the browser after completion.
